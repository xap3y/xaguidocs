## Gradle installation

<a href={"https://central.sonatype.com/artifact/eu.xap3y/xagui"}>
    <img alt="Maven Central Version" src="https://img.shields.io/maven-central/v/eu.xap3y/xagui" />

```groovy
repositories {
    mavenCentral()
}
```

### Add the dependency

```groovy
dependencies {
    implementation 'eu.xap3y:xagui:1.1'
}
```

### Shading

Replace `your.plugin.utils` with your package name

```groovy
plugins {
    id 'com.github.johnrengelman.shadow' version '8.1.1'
    id 'java'
}

tasks {
    shadowJar {
        relocate 'eu.xap3y.xagui', 'your.plugin.utils'
    }
}
```
