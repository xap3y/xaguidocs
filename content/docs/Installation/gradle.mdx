## Gradle installation


### First, add JitPack repository to your build file

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}
```

### Then, add the dependency

| Replace `VERSION` with the version you want to use or with commit hash

```groovy
dependencies {
    implementation 'com.github.xap3y:xagui:VERSION'
}
```

### Shading

Replace `your.plugin.utils` with your package name

```groovy
plugins {
    id 'com.github.johnrengelman.shadow' version '8.1.1'
    id 'java'
}

tasks {
    shadowJar {
        relocate 'eu.xap3y.xagui', 'your.plugin.utils'
    }
}
```
